Get Started with
Axiom Code

The elite agentic coding suite by Ruphak Varmaa. Professional setup in minutes.

1. Prerequisites

Ensure you have Node.js 18.0.0 or higher installed on your system. Axiom Code is optimized for high-performance terminal environments.

Terminal node --version

2. Global Installation

Install Axiom Code globally using npm. This gives you access to the axiom command anywhere on your machine.

Terminal npm install -g axiom-code@latest
Tip: Always include @latest to ensure you have the most recent "Three Power" updates by Ruphak Varmaa.

3. Initial Configuration

Launch the CLI for the first time. Axiom will detect no profiles and guide you through adding your first LLM provider (Anthropic, OpenAI, or any specialized endpoint).

Terminal axiom

Follow the interactive prompts to enter your API key and set your default model (e.g., claude-3-5-sonnet).

4. Usage Patterns

Axiom Code supports three primary modes of operation:

Interactive REPL

Best for deep architectural work and multi-step tasks.

axiom

One-Shot Commands

Perfect for quick edits or code analysis.

axiom "Find all uses of useEffect and refactor to use a custom hook"

Standard Output Injection

Axiom can read from pipes to process logs or diffs.

git diff | axiom "Explain these changes and write a commit message"

5. Elite Features

Once setup, you can leverage the advanced suite directly through slash commands:

  • /watch <cmd> — Activate Guardian Mode for proactive error correction.
  • /mcp — Connect to any Model Context Protocol server for dynamic tool-building.
  • /pr — Generate high-fidelity AI Pull Requests.
